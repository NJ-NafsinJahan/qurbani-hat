## ✨ Qurbani Hat: Halal and Safe Qurbani.

## 📑 Description

A modern livestock marketplace where people can explore livestock for Qurbani such as cows,goats,sheep and camel. It producing highest quality and standard livestock in markets.

Users can view details and place a booking after authentication.

## 🚀 key features of this project

🐄 1. Livestock Listing & Details

Browse a wide range of cattle, goats, sheep, and camels with detailed information including price, breed, weight, and location.

🔍 2. Dynamic Details Page

Each animal has a dedicated details page with full information and status (Available / Booked).

🔐 3. Authentication System (UI Ready)

User can register, login, and access personalized features.

🟢 4. State & Conditional Rendering

Handled UI states (error, success, status badge) using conditional rendering.

⚡ 5. Responsive & Modern UI

Built with Tailwind CSS ensuring a clean, fast, and mobile-friendly user experience.

🌐 6. Live Data Integration

Fetches livestock data dynamically from a hosted JSON API.

## ⚙️ Technologies Used

- Next.js
- React.js
- Tailwind CSS, DaisyUI
- JavaScript (ES6+)
- JSON (for livestock data)
- Better-auth, MongoDB
- Google Authentication
